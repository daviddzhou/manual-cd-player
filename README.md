## What Is This?

This is **the official manual CD player widget** designed exclusively for **memelord.com/docs** — where keyboard warriors become DJ legends.

Unlike those weak algorithmic streaming services, this CD player respects the grind. If you want to hear the audio, you gotta **earn it**. Spin that disc like you're back in 2001 ripping Napster downloads.

## Features

- **Manual Spin Control**: Drag the CD to spin it like a real turntable. No autoplay. No shortcuts.
- **Coast Mode Toggle**: Switch between realistic physics (CD keeps spinning) or instant stop for precision control
- **RPM Controls**: Fine-tune your min/max RPM thresholds. Too slow? Audio stops. Too fast? Chipmunk mode.
- **Real-time Status**: LED-style display showing your current RPM and track progress
- **Variable Speed Audio**: Playback rate scales with your spin speed. The faster you spin, the faster it plays.
- **Windows XP Aesthetic**: Matches the legendary memelord.com retro theme
- **Touch Support**: Works on mobile for on-the-go DJ sessions

## How to Use

1. **Navigate** to the widget in the bottom-right corner
2. **Drag** the CD to spin it manually
3. **Hit minimum RPM** to start audio playback
4. **Maintain speed** to keep the groove going
5. **Experiment** with Coast Mode and RPM settings
6. **Become a legend**
